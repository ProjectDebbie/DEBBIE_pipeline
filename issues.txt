Rule added 
DEBBIE ONTOLOGY over UMLS,  when the same term is mapping with DEBBIE Ontology and with UMLS the UMLS 
label is delete.  Only if is the same term or if the DEBBIE term is bigger than UMLS. 

Ejemplos de prioridad:

documento pdo1

PDO:           Biomaterial (DEBBIE) -  BiologicallyActiveSubstance (UMLS)
capsule:       Structure   (DEBBIE) -  Biomaterial                 (UMLS)

collagen type ---> mapped as 
BiologicallyActiveSubstance by DEBBIE only collagen ---> UMLS collagen type
